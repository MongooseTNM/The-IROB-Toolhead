# The-IROB-Toolhead
## AKA The I-Ripped-Off-Burgo Toolhead
Basically a redesign of the Takeoff Toolhead by Kizime123 

Specs:
 - Dual 4028 fans
 - Dragon UHF
 - Sherpa Mini compatibility
 - 3010 heatbreak fan
 - Custom Klicky "tap" solution
 - 6mm belts
